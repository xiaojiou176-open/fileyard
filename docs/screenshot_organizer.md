## Screenshot Organizer With Review Queue And Dry-Run Safety

Screenshot cleanup is usually repetitive, but the mistakes still matter. Work images, receipts, and random references all end up mixed together, so a “sort automatically now” button is often the wrong answer.

Fileorganize treats screenshot cleanup like a review workflow instead of a blind rename script.

## What Changes

- Analyze drafts titles and categories first
- Review Queue highlights the rows that still need judgment
- Rule drafts and learned suggestions reduce repeated screenshot decisions
- Dry-run Apply shows the plan before execution

## Good Fit

- desktop dump folders
- recurring screenshot cleanup for work or research
- mixed screenshot + PDF + photo batches

## Not A Fit

- autonomous cleanup with no inspection
- hosted screenshot knowledge-base products
